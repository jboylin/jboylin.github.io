The window cleaning app: Cleanr

Web-Based Window Cleaning.

Cleanr is an example mobile app running on both IOS and Android to help connect window cleaners with clients and facilitate payment in-app.

to accomplish this it uses the following technology:

  • React Native
  • Expo
  • Node.js
  • Axios
  • React Native Paper
  • Google Firebase
  • Google Maps
  • Stripe
